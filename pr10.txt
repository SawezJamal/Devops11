Program 10 — Azure Pipelines with Self-Hosted Agent (Clean Correct Steps)

We will do this from scratch in the safest order.

WHAT YOU NEED BEFORE STARTING

Verify these work first:

java -version
mvn -version
git --version

Also:

GitHub account
Azure DevOps account
PART 1 — CREATE MAVEN PROJECT
STEP 1 — Create Folder

Open Command Prompt:

mkdir program10
cd program10
STEP 2 — Generate Maven Project

Run:

mvn archetype:generate -DgroupId=com.example -DartifactId=azure-pipeline-project -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false
STEP 3 — Move Into Project
cd azure-pipeline-project
STEP 4 — Replace App.java

Go to:

src/main/java/com/example/

Open App.java.

Replace with:

package com.example;

public class App {

    public static void main(String[] args) {

        System.out.println("Azure Pipeline CI/CD Project");

        int a = 5;
        int b = 10;

        System.out.println("Sum = " + (a + b));
    }
}

Save file.

STEP 5 — Test Project Locally

Run:

mvn clean test

Expected:

BUILD SUCCESS
PART 2 — PUSH PROJECT TO GITHUB
STEP 6 — Create GitHub Repository

Open:

GitHub

Create repository:

azure-pipeline-project

Do NOT add README.

STEP 7 — Push Project to GitHub

Inside project folder:

git init
git add .
git commit -m "Initial commit"

Add GitHub repo:

git remote add origin https://github.com/<your-username>/azure-pipeline-project.git

Push:

git branch -M main
git push -u origin main
PART 3 — CREATE AZURE DEVOPS PROJECT
STEP 8 — Open Azure DevOps

Azure DevOps Portal

STEP 9 — Create Project

Click:

Project Name	Lab-program-10

PART 4 — CREATE SELF-HOSTED AGENT
STEP 10 — Create Agent Pool

Go to:

Project Settings

↓

Agent Pools

↓

Add Pool

Select:

Self-hosted

Pool name:

MyLocalPool

STEP 11 — Download Agent

Open created pool.

Click:

New Agent

Choose:

Windows

Download ZIP.

STEP 12 — Extract Agent

Extract ZIP to:

C:\azureagent
PART 5 — CREATE PAT TOKEN
STEP 13 — Create PAT

In Azure DevOps:

Click profile icon →

Personal Access Tokens

↓

New Token
STEP 14 — PAT Settings

Use:

Field	Value
Name	MyAgentToken
Expiration	90 days
Access	Full access

Click:

Create

Copy token immediately.

cd C:\azureagent

config.cmd

MyLocalPool

Listening for Jobs

PART 7 — CREATE AZURE PIPELINE
STEP 20 — Create Pipeline

Go to:

Pipelines

↓

New Pipeline
STEP 21 — Choose GitHub

Select:

GitHub

Authorize if asked.

Select repository:

azure-pipeline-project
STEP 22 — Choose Starter Pipeline

Select:

Starter Pipeline
STEP 23 — Replace YAML

Delete existing YAML.

Paste:

trigger:
- main

pool:
  name: 'MyLocalPool'

steps:

- checkout: self
  displayName: 'Checkout Code from GitHub'

- script: mvn clean test
  displayName: 'Build and Run Unit Tests'

- task: PublishTestResults@2
  inputs:
    testResultsFiles: '**/target/surefire-reports/TEST-*.xml'
    testResultsFormat: 'JUnit'
    failTaskOnMissingResultsFile: true
  displayName: 'Publish Maven Test Results'

- task: PublishBuildArtifacts@1
  inputs:
    PathtoPublish: 'target'
    ArtifactName: 'drop'
    publishLocation: 'Container'
  displayName: 'Publish Build Artifacts'
STEP 24 — Run Pipeline

Click:

Run

source code checkout

artifacts
