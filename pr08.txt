Step 11: Prepare Ansible

Open Ubuntu

Create directory

mkdir ~/ansible-deploy

cd ~/ansible-deploy

Inventory

nano hosts.ini
[local]
localhost ansible_connection=local

Save.

Create deployment directory

sudo mkdir -p /opt/deployment

sudo chmod 777 /opt/deployment
Step 12: Create Playbook
nano deploy.yml

For Windows Jenkins, use the Windows workspace through /mnt/c. For example:

---
- name: Deploy Maven Artifact
  hosts: local

  tasks:
    - name: Copy Maven artifact
      copy:
        src: "/mnt/c/ProgramData/Jenkins/.jenkins/workspace/HelloMaven-CI/target/HelloMaven-1.0-SNAPSHOT.jar"
        dest: "/opt/deployment/HelloMaven.jar"

If your Jenkins workspace is in a different location, replace the src path with the actual workspace path.

Step 13: Test Playbook
ansible-playbook -i hosts.ini deploy.yml

Verify

ls -l /opt/deployment

Expected

HelloMaven.jar
