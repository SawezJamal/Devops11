Step 1: Install WSL on Windows

Open PowerShell as Administrator and run:

wsl --install

Restart the system if prompted.

Check available distributions:

wsl --list --online

Install Ubuntu:

wsl --install Ubuntu-24.04

After installation, restart the PC.

Step 2: Launch Ubuntu

Open PowerShell and run:

wsl

Create a Linux user account:

Enter new UNIX username: sawez
New password: ********

Ubuntu terminal opens:

sawez@hostname:~$
Step 3: Update Ubuntu Packages
sudo apt update
sudo apt upgrade -y
Step 4: Install Ansible
sudo apt install ansible -y

Verify installation:

ansible --version

Sample Output:

ansible [core 2.x.x]
python version = 3.x.x
Step 5: Create Working Directory
mkdir Program7
cd Program7
Step 6: Create Inventory File

Create inventory file:

nano inventory.ini

Add:

[local]
localhost ansible_connection=local

Save:

Ctrl + O
Enter
Ctrl + X

Verify:

cat inventory.ini
Step 7: Test Connectivity Using Ping Module

Command:

ansible all -i inventory.ini -m ping

Output:

localhost | SUCCESS => {
    "changed": false,
    "ping": "pong"
}
Step 8: Create Ansible Playbook

Create playbook:

nano playbook.yml

Add:

---
- name: First Playbook
  hosts: all

  tasks:
    - name: Display Message
      debug:
        msg: "Hello from Ansible"

Save and exit.

Step 9: Execute Playbook

Run:

ansible-playbook -i inventory.ini playbook.yml

Output:

PLAY [First Playbook]

TASK [Display Message]
ok: [localhost] => {
    "msg": "Hello from Ansible"
}

PLAY RECAP
localhost : ok=1 changed=0 failed=0
Important Commands
ansible --version
ansible all -i inventory.ini -m ping
ansible-playbook -i inventory.ini playbook.yml
cat inventory.ini
cat playbook.yml
